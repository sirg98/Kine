# Clínica Kinesiología - Estructura de Archivos

- `index.php`: Página de inicio pública
- `partials/`: Navbar y footer
- `styles/`: CSS global
- `assets/img/`: Imágenes
- `components/`: Código compartido, base de datos
- `doctor/crear_usuario.php`: Solo el doctor puede crear usuarios
- `auth/login.php`: Login de usuarios
- `auth/logout.php`: Logout de usuarios
- `paciente/`: (dashboard y archivos del paciente)

Asegúrate de actualizar los includes/rutas en los archivos según la nueva estructura.
